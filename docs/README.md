### How to run?🏃
```bash
    # In your git bash, clone the repo.
    git clone https://github.com/angelo-francisco/WAPP-WeatherAPP.git

    # Activate the enviroment
    ./venv/Scripts/Activate.ps1

    # Go to the source
    cd src

    # run the project
    python manage.py runserver
```
